# ForestRx 개발 요약

## 서비스 목표

사용자의 현재 위치, 체력, 경험, 컨디션, 이동 가능 시간, 날씨 안전도를 종합해 오늘 가장 안전하고 적합한 등산 코스를 추천한다.

## 핵심 기능

- 사용자 상태 입력: 연령대, 등산 경험, 컨디션, 활동 강도, 이동 가능 시간, 동반인
- GPS 위치 기반 추천: 주변 산과 접근 시간을 계산해 후보 코스 제공
- Top3 추천: 체력 적합도, 날씨 안전도, 접근성 점수를 합산
- 날씨 안전 분석: 강수량, 풍속, 기온, 일몰 시간 기준 위험도 제공
- 대체 코스 추천: 위험 요소가 있으면 더 짧고 쉬운 코스 우선 제안
- 지도 시각화: 추후 Kakao/Naver Maps 연동 예정

## 1차 MVP

외부 API 연동 전에도 개발과 시연이 가능하도록 샘플 코스 데이터와 모의 날씨 데이터를 사용한다.

1. 상태 입력 폼
2. 브라우저 Geolocation 위치 수집
3. Django 추천 API
4. Vue 추천 결과 카드
5. 코스 상세 안전 분석

## API 초안

- `GET /api/health/`
- `GET /api/courses/`
- `POST /api/recommendations/`

## 추천 점수

- 체력 적합도 35%
- 날씨 안전도 30%
- 접근성 20%
- 시간 적합도 15%

점수는 0-100 사이로 정규화한다.
